# Research Notes

This folder holds external research documents and comparative notes that inform Moltark's architecture.

Read these when you need:

- prior art on project bootstrapping and update-safe generators
- examples of module systems, extension APIs, and repository reconciliation
- lessons from real tools Moltark may want to emulate or deliberately avoid

## Documents

- [Rust Project Bootstrapping Research](./rust-project-bootstrapping-research.md)
- [Go Project Scaffolding Research](./go-project-scaffolding-research.md)
- [Bazel Gazelle Research](./bazel-gazelle-research.md)

## Conventions

When adding new research notes:

- use kebab-case file names ending with `-research.md`
- start with a clear title
- add a short metadata preamble with date and scope
- keep the document explicit about sources and relevance to Moltark
